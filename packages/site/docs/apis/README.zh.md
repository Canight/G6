## 参与开发

```
# 在项目的根目录安装依赖

$ pnpm install

# 从项目根目录进入到 g6 包文件目录下
$ cd packages/g6

# 构建
$ npm run build

# 启动集成测试 demo
$ npm run dev

# 运行所有单元测试
$ npm run test

# 运行官网，在此之前请确保 g6 包已经构建
$ cd packages/site
$ tnpm start

```
