平面，默认躺在 XZ 平面上

## width

**类型**：`number`

**默认值**：`0`

**是否必须**：false

**说明**：宽度

## depth

**类型**：`number`

**默认值**：`0`

**是否必须**：false

**说明**：深度

## widthSegments

**类型**：`number`

**默认值**：`1`

**是否必须**：false

**说明**：宽度分段数

## depthSegments

**类型**：`number`

**默认值**：`1`

**是否必须**：false

**说明**：深度分段数
